Course Projects and Class Demos

Lab 5: SIS Web Portal

This project is a Student Information System web portal built with a simple web interface and a SQLite database. It supports students, faculty, courses, enrollments, drops, and grade submission.

  • Add students, faculty, and courses
  • Enroll students into courses
  • Drop students from courses
  • Submit grades for enrolled students
  • View seats remaining and student transcripts

This project helped me understand transactional database design, primary keys, foreign keys, and many-to-many relationships using an enrollments bridge table.

Lab 6: SIS Enrollment BI Dashboard

This project is a business intelligence dashboard for analyzing SIS enrollment data. It uses descriptive analytics and OLAP-style filtering to summarize enrollment activity by term, department, and course.

  • Shows total enrollments for a selected term
  • Calculates average fill rate
  • Calculates course drop rate
  • Displays average grade performance
  • Includes course-level and department-level summaries

This project helped me understand the difference between transactional tables and data warehouse summary tables for reporting.

AI Finger Calculator Demo

This class demo uses AI and computer vision concepts to create an interactive finger-counting calculator. The application uses a camera to detect hand gestures and perform calculator actions.

  • Uses real-time camera input
  • Detects fingers and hand gestures
  • Supports calculator-style interaction
  • Shows how AI can be integrated into a user interface

This demo helped me see how AI can move beyond theory and become part of practical, interactive applications.
